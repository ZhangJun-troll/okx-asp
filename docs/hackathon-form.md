# OKX-AI-Genesis Hackathon — Submission Form

## Project Name
**OKX-ASP: Agentic Service Provider**

## Project Description (280 chars)
Multi-Agent crypto sentiment intelligence on X-Layer. 4 AI agents (Dispatcher, Collector, Analyst, Settlement) form an autonomous economy with X402 micropayments. Real-time news crawling, sentiment scoring, and on-chain settlement.

## Category
AI + DeFi / Agent Economy

## Technical Architecture
- **Backend**: Python 3.11 + FastAPI microservices
- **Smart Contracts**: Solidity 0.8.19 on X-Layer (Chain ID 196)
- **Agent Communication**: REST API + A2A message protocol
- **Payment**: X402 protocol simulation with on-chain settlement
- **Data Sources**: RSS feeds (CoinDesk, CoinTelegraph, Decrypt) + whale TX monitor
- **Frontend**: Zero-dependency HTML/CSS/JS

## Key Technical Innovations
1. **A2A Autonomous Economy**: Agents charge each other for services via X402 micropayments
2. **4-Agent Cluster**: Specialized agents with independent economic incentives
3. **X-Layer Native**: All settlements on-chain, sub-cent transaction costs
4. **Transparent Ledger**: Every A2A transaction recorded with block numbers

## OKX Ecosystem Integration
- X-Layer L2 deployment (Chain ID 196)
- OKX Agent Payment Protocol for subscriptions
- OKX DEX API integration (planned) for whale monitoring
- OKX Wallet for user authentication

## Business Model
- Free tier: Daily sentiment summary (5 coins)
- Basic: 0.005 ETH/month (deep reports, risk alerts)
- Premium: 0.02 ETH/month (full analysis, 30 reports)
- Platform fee: 2% on A2A transactions
